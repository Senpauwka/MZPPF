# МЗППФ

## Министерство защиты прав участников флудов

Официальный сайт организации МЗППФ.

Цель проекта — создание независимой системы защиты участников флудов, рассмотрения конфликтных ситуаций, анализа обращений и формирования прозрачной системы решений.

---

# 📌 О проекте

МЗППФ занимается:

- рассмотрением обращений участников;
- независимым анализом конфликтных ситуаций;
- проверкой спорных решений администрации;
- защитой прав участников;
- созданием архива решений и прецедентов.

Основной принцип:

> Справедливость выше власти.

# 📜 Документы

В проекте используются:

## Устав МЗППФ

Определяет:

- цели организации;
- основные принципы;
- права участников;
- ограничения деятельности.

---

## Внутренний регламент

Определяет:

- порядок работы сотрудников;
- процесс рассмотрения обращений;
- уровни допуска;
- внутренние правила.

---

# 👥 Структура сотрудников

Планируемая система:

- Руководящий состав;
- Администрация;
- Сотрудники рассмотрения обращений;
- Стажёры.

---

# 📝 Обращения

Будущая система обращений позволит:

- создавать дела;
- получать номер обращения;
- отслеживать статус;
- получать решения;
- сохранять историю.

---

# 🤖 Будущая интеграция Telegram

Планируется создание Telegram-бота МЗППФ.

Основные функции:

- идентификация пользователей;
- авторизация сотрудников;
- управление уровнями допуска;
- работа с обращениями;
- уведомления;
- ведение истории действий.

---

# 🔐 Система доступа

Планируемые уровни:

L0 — Стажёр
L1 — Сотрудник
L2 — Специалист рассмотрения
L3 — Руководитель направления
L4 — Руководящий состав
